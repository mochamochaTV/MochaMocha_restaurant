📁 もっちゃもっちゃ食堂（プロジェクトの親フォルダ）
├── 📁 assets            ← 画像や音声をまとめるフォルダ
│   ├── 📁 img           ← すべての .png をここに入れる
│   └── 📁 sounds        ← すべての .mp3 をここに入れる
├── 📁 css
│   └── 📄 style.css     ← 新しく作るCSSファイル
├── 📁 js
│   ├── 📄 data.js       ← 単語やガチャなどのデータ一覧
│   ├── 📄 audio.js      ← 音声の読み込みと管理
│   └── 📄 main.js       ← ゲームを動かすメインのプログラム
└── 📄 index.html        ← これを開いてゲームを起動する

#画像更新
git add .
git commit -m "画像差し替え"
#この後にCommit&Push

# 1. 変更したファイルをステージング（確定の準備）に追加
git add .

# 2. 変更内容に名前をつけて確定（コミット）
git commit -m "ホーム画面の背景とレイアウトを更新"

# 3. （GitHubなどに同期している場合）サーバーへ送信
git push
